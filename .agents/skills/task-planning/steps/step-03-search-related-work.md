# Step 03: Search Related Work

## Purpose

Ground planning in prior repo decisions and related documentation.

## Entry Criteria

- Step 02 exit criteria are met

## Actions

1. Identify search terms from the task, goals, and components.
2. Search related tasks, HLDs, PRDs, domain-analysis content, and code where relevant.
3. Read the matches that materially constrain the plan.
4. Summarize reusable patterns, constraints, and remaining gaps.

## Expected Outputs

- search term list
- related-work summary
- unresolved gaps after research

## Exit Criteria

- planning is grounded in the most relevant existing work
- reusable patterns and constraints are explicit

## Confirmation Gate

Present the findings and ask for confirmation before Step 04.
