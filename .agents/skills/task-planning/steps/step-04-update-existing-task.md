# Step 04: Update Existing Task with Research Findings

## Purpose

Keep the task accurate without overloading it with technical detail.

## Entry Criteria

- Step 03 exit criteria are met

## Actions

1. Compare the task against research findings.
2. Update the task only if research proves the problem statement, goals, or acceptance criteria are materially wrong or incomplete.
3. Defer technical planning detail to the plan.

## Expected Outputs

- updated task only if required
- explicit note if no update was needed

## Exit Criteria

- the task remains product-readable
- only corrections to task truth were applied

## Proceed Condition

Proceed to Step 05 once the task accurately reflects researched requirements.
